# Patrick's School Fundraising Site

## What This Is
A simple GitHub Pages fundraising website for Patrick Musemeza's nursery school for orphan refugees in Uganda. The school serves children from Congolese refugee families in a church congregation of ~300 members (~150 children, many orphans).

## Key Details
- **Hosted on**: GitHub Pages as a project repo (username.github.io is reserved for Sam's personal site)
- **Source materials**: The `whatsapp/` folder contains raw WhatsApp messages and photos from Patrick. This folder is gitignored and should NOT be committed.
- **Problem statement**: Taken verbatim from Patrick's WhatsApp message timestamped 2026/3/14 11:29. Do not edit the quoted text.
- **Funding need**: 1,200,000 UGX/month for 4 teacher salaries, for 2 years
- **Donation flow**: US donors give to Sam, who handles international transfer to Patrick

## Site Structure
- `index.html` — single-page site with problem statement, photos, and donation info
- `images/` — curated subset of photos from WhatsApp (group photos + select portraits)
